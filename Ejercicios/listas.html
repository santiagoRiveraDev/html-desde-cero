Lenguajes para el Desarrollo Web

HTML(Hipertext Markup Language):
Es un lenguaje de marcado que se utiliza para estructurar datos en una pagina web.
CSS(Cascade Style Sheet):
Es un lenguaje de marcado que se utiliza para dar estilos en una pagina web.
JS(Javascript):
Es un lenguaje de programacion que se utiliza para dar interactividad a una pagina web.
